Entidades / Modelos do Projeto

1. Appointment (Agendamento)
- id: number
- nome: string
- email: string
- telefone: string
- servico: string
- data: string
- hora: string
- mensagem: string (opcional)
- status: string ('pending' | 'confirmed')

2. Service (Serviço)
- key: string
- label: string
- price: string
- active: boolean

3. StaffMember (Equipe/Admin)
- email: string
- password: string
- name: string
- role: string ('Psicóloga' | 'Psicólogo' | 'Admin')

4. AvailableTime (Horário disponível)
- date: string
- times: string[]

5. PsychologistProfile (Perfil de psicólogo/a)
- name: string
- title: string
- crp: string
- image: string
- bio: string
- education: string[]
- specializations: string[]
- experience: string[]
- approaches: string[]

6. CurrentUser (Usuário admin autenticado em sessão)
- email: string
- password: string
- name: string
- role: string

7. Payment (Pagamento)
- paymentId: number
- appointmentId: number
- patientId: number
- amount: string
- method: string ('credit_card' | 'pix' | 'boleto' | 'cash')
- status: string ('pending' | 'paid' | 'failed')
- date: string
- insuranceUsed: boolean
- healthPlanId: number (opcional)

Exemplos de Payment:
1. paymentId: 301, appointmentId: 501, patientId: 101, amount: 'R$ 120', method: 'pix', status: 'paid', date: '2026-06-10', insuranceUsed: false, healthPlanId: null
2. paymentId: 302, appointmentId: 502, patientId: 102, amount: 'R$ 80', method: 'credit_card', status: 'paid', date: '2026-06-18', insuranceUsed: true, healthPlanId: 202
3. paymentId: 303, appointmentId: 503, patientId: 104, amount: 'R$ 180', method: 'boleto', status: 'pending', date: '2026-06-15', insuranceUsed: true, healthPlanId: 203

8. HealthPlan (Plano de saúde)
- healthPlanId: number
- provider: string
- planType: string
- coverageLevel: string
- validUntil: string
- policyNumber: string
- holderName: string
